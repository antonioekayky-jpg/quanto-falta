#  Quanto Falta?

Uma aplicação web simples e funcional que calcula quanto tempo falta para atingir um objetivo financeiro com base no valor que o usuário consegue guardar por mês.

O usuário informa quanto consegue guardar mensalmente e define um objetivo (ex: comprar um celular, carro, viagem, etc). O sistema calcula o tempo estimado necessário para atingir esse objetivo.

Além disso, o projeto permite salvar múltiplos objetivos e acompanhar o progresso ao longo do tempo.

---

##  Funcionalidades

- Inserir valor que pode guardar por mês
- Definir um objetivo financeiro
- Calcular o tempo necessário para atingir o objetivo
- Exibir resultados de forma clara
- Salvar múltiplos objetivos
- Listar objetivos salvos
- Interface simples e responsiva

---

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- (Opcional) Firebase para armazenamento de dados

---

##  Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Organização de código em HTML, CSS e JavaScript separados
- Manipulação do DOM
- Validação de dados
- Estruturação de aplicações web simples
- Integração com banco de dados (Firebase)

---

##  Como usar

1. Abra o arquivo `index.html` no navegador
2. Digite quanto consegue guardar por mês
3. Defina seu objetivo
4. Clique em **Calcular tempo**
5. Veja quanto tempo falta para atingir sua meta
